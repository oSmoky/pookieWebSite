# Angelina Wrapped

React/Vite сайт-подарок: снежно-синий cinematic Wrapped, физический canvas в hero, галерея с tilt-эффектом, счетчик отношений, письмо, финальный сюрприз и музыка, которая меняется при скролле.

## Запуск

```bash
npm run dev
```

## Фото

Файлы лежат в `public/photos`:

- `cover.jpg` - главная фотография.
- `photo-01.jpg`, `photo-02.jpg`, `photo-03.jpg` и дальше - карточки галереи.

Подписи меняются в `src/data/siteData.js`, массив `photos`.

## Музыка

Mp3-файлы лежат в `public/audio`:

- `01-behind-blue-eyes.mp3`
- `02-nutshell.mp3`
- `03-wish-you-were-here.mp3`
- `04-toxicity.mp3`
- `05-leave-out-all-the-rest.mp3`
- `06-otherside.mp3`

Браузер запускает звук только после первого клика по кнопке `Открыть`. Потом треки переключаются по секциям.
